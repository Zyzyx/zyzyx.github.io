Yum Repository

2011-10-02

For my non-work side projects I find myself packaging software for Fedora from time to time. I created a yum repository for those packages over here. So far it doesn’t contain anything I want to brag about, just dependencies for projects I’m getting involved with. I’ll post more about those when I reach some kind of appreciable goal. MetaRPG might end up in there some day too.

For the Fedora-novice, take that text file and save it in /etc/yum.repos.d. Once saved you can use repoquery (from the yum-utils package) to find out what it is providing.
